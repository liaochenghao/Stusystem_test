```
    PUT/PATCH     /auth/user/score_detail/[score_detail_id]/
```

```
{
    "department": ""        str     收件人部门   最大长度30
    "phone":    ""          str     收件联系电话 最大长度30
    "country": ""           str     收件国家    最大长度30
    "post_code": ""         str     邮编       最大长度30
    "address"：""           str      收件详细地址  最大长度60
}
```
